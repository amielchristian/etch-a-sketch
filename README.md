# Etch a Sketch
This is a simple web app for drawing on a 480x480 grid. You can access the website [here](https://amielchristian.github.io/etch-a-sketch).

This project is not yet complete. Simply hovering over the grid paints over it, and other features have yet to be implemented. Future changes will address these flaws.
